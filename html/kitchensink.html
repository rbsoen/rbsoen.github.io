rw-auto

rw-auto provides a breathing space for the page.

In mobile displays, this will add a tiny bit of margin to the left and right of the page.

In desktop displays, this will "clip" the page on a set maximum width, leaving whitespace on the sides. This "opens up" the overall design and helps readability.

Responsive Classes (precise)

Using these precise classes you can control the level of responsiveness in each screen size.

Class names rw-col s* will always apply.

Class names rw-col m* will apply at width ≥ 600px.

Class names rw-col l* will apply at width ≥ 992px.
